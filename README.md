# Courier Tracking Automation System

A full-stack courier tracking dashboard that automates tracking, status updates, Excel uploads, and daily email reports for multiple courier services.

# Currently supported:

Professional Couriers

Franch Express

# Features
✅ Multi-Courier Support

Separate handling for Professional Courier and Franch Express

Independent Excel uploads & tracking logic

Separate database tables per courier

✅ Tracking Automation

Fetches live tracking status from courier portals

Updates:

Delivery status

Last status text

Last checked timestamp (IST)

✅ Daily Email Report (Automated)

Runs via scheduler (APScheduer)

Sends HTML email with tables

Two sections:

Professional Courier – Pending

Franch Express – Pending

IST-formatted dates

Optional manual trigger endpoint
